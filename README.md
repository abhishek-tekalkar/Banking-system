# BankFusion

The Banking System application is designed to manage customer accounts and facilitate core banking operations. The system includes features for account creation, deposit and withdrawal transactions, balance inquiries, and PIN changes. The user-friendly interface, built with Swing, ensures that the application is easy to navigate while maintaining robust security.
